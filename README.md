# TypeScript Project

This is a starter TypeScript project. It includes a basic configuration and a sample source file.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Build the project:
   ```sh
   npm run build
   ```
3. Run the project:
   ```sh
   npm start
   ```

## Project Structure
- `src/` - Source TypeScript files
- `dist/` - Compiled JavaScript output
- `tsconfig.json` - TypeScript configuration
- `package.json` - NPM scripts and dependencies
